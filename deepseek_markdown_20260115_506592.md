# Health & Wellness Hub

A comprehensive health and wellness website with multiple interactive features including nutrition guides, fitness plans, mental health resources, and contact forms.

## Features

### 1. Multi-Page Navigation
- Home page with overview
- Nutrition guide with meal plans
- Fitness programs with calorie calculator
- Mental wellness resources with assessment
- Contact page with form

### 2. Interactive Calculators
- Hydration calculator (water intake based on weight)
- Calorie burn calculator (activity-based)
- Mental health assessment

### 3. Multimedia Content
- Audio players for guided meditation and nutrition tips
- Responsive image galleries
- Interactive tables

### 4. User Features
- Local storage for saving preferences
- Form validation
- Responsive design for all devices
- Smooth animations and transitions
